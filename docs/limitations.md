# Limitations

This repository describes an architectural proposal and early scaffold.

## Current Limitations
- No production-grade elicitation pipeline is implemented yet.
- No empirical benchmark results are provided.
- Reasoning and consensus methods are placeholders.
- Domain examples are illustrative only.

## Research Risks
- Expert elicitation can inherit bias from participant selection.
- Confidence estimates can be inconsistent across experts.
- Structured models may oversimplify ambiguous real-world contexts.

## Scope Note
This project is intended as a neutral technical exploration, not as evidence of superior performance over large-scale model pretraining approaches.
