# Prior Art

This project builds on established lines of research rather than claiming these ideas are new.

## Foundations
- Expert Systems (1970s–1990s)
- Knowledge Engineering / Knowledge Elicitation
- Knowledge Graph AI
- Human Feedback systems (RLHF)

## Positioning
This repository proposes a modern architecture that combines:

- expert knowledge elicitation
- structured domain knowledge graphs
- multi-expert consensus modeling
- modern AI-assisted knowledge extraction

The intent is to synthesize these complementary approaches into a practical, transparent, expert-driven AI design.
